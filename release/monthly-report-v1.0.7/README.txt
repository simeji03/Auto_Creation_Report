    月報作成支援ツール v1.0.7 - 3ステップで簡単起動！
====================================================================

【必要なもの】
- Docker Desktop（必須）
- OpenAI APIキー（AI機能を使う場合）

【起動方法】たったの3ステップ！

■ Windows の場合：
1. manual-start.bat をダブルクリック
2. ブラウザで http://localhost:3456 を開く
3. 設定画面でOpenAI APIキーを入力

■ Mac の場合：
1. manual-start.command を右クリック → 「開く」
2. ブラウザで http://localhost:3456 を開く  
3. 設定画面でOpenAI APIキーを入力

■ Linux の場合：
1. ターミナルで ./manual-start.sh を実行
2. ブラウザで http://localhost:3456 を開く
3. 設定画面でOpenAI APIキーを入力
